Como funciona (nuvem)
  1. Você escolhe 1 placa ou todos, no celular
  2. O servidor consulta o rastreador (Sitrax)
  3. Arquivos brutos ficam em pasta temporária no servidor
  4. O servidor monta 1 PDF de resumo (cidade / horário)
  5. Os arquivos brutos são apagados
  6. Você baixa só o resumo, se quiser
